Our Services


		Nowadays, In our country traffic jam is a mysterious thing. General mass suffer
		a lot for traffic jam. There are many purposes for traffic jam. One of them is
		the manual tolling system in bridges, flyovers, express-way etc. If we will 
		build a system that will help these vehicle to pre-pay all of these toll via 
		online then this problem may be disappeared. We also have a purpose to 
		categorize all of these vehicles. We create a database to all of these traffic
		which are available on these certain routes. When a vehicle owner pay its
		vehicle taxes then we check the information about vehicles fitness, license
		validity, driving license. If any of these information will negative then we
		don’t give the permission to cross this. That’s help us to avoid road accident
	    and build up discipline transportation system in Bangladesh.

How you can pay tax in this system

1. Firstly register the vehicles with all details information. These information are stored in our database.
2. If any of these vehicles license validity is expired or banned then license renewed from this website.
3. All of the bridges & flyovers toll are payable by this website.
4. Authority: When any vehicles want to pay its transport toll then administration panel automatically check the vehicles fitness, engine capability, license duration & driver information. If any of these criteria is disappeared then administration panel banned this vehicle.
